## Project Breakdown

### Bookmark Saver

Persistent bookmark management system using LocalStorage with structured data modeling and input validation.

### Colour Palette Generator

Dynamic color generation utility leveraging event-driven UI updates and clipboard integration.

### Password Generator

Secure password generator implementing configurable entropy rules and character set logic.

### Expense Tracker

Client-side financial tracker implementing transactional state management, dynamic rendering, and persistent storage.

### Form Validator

Reusable validation engine utilizing regex, real-time error messaging, and controlled input states.

### Kanban Board

Drag-and-drop task management interface using native Drag and Drop API with synchronized UI state.

### Quiz Game

Interactive quiz engine implementing dynamic question rendering, score tracking, and conditional UI flows.
